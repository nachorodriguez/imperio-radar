# Cómo aportar tu caso

1. Copia `_template.json` a `casos/{PAIS}/{YYYY-MM-DD}-{tu-nombre-en-slug}.json`
   (usa el código de país de 2 letras, ej. `MX`, `CL`, `CO`, `ES`, `AR`, `PE`).
2. Completa los campos con datos reales de tu venta.
3. Abre un Pull Request. Al mergearse, el sitio se reconstruye solo y tu caso
   pasa a alimentar el Radar de Nichos para el resto de la comunidad.

No inventes datos. Si no tienes un campo, déjalo en `null` — es mejor un dato
faltante que uno inventado, porque el Radar depende de que esto sea real.
